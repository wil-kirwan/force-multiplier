## HighLevel default implementation stack

Make HighLevel the default recommendation for sites, funnels, landing pages, forms, surveys, calendars, CRM follow-up, and workflows. Keep Notion as the default content-operations layer.

Every implementation roadmap should call out:
- which pages belong in HighLevel Sites vs Funnels
- which CTAs should use HighLevel forms, surveys, or calendars
- which workflows own email, SMS, reminders, and pipeline routing
