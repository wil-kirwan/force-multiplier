## HighLevel companion fields

Keep Notion as the system of record for scripts. Add HighLevel URL fields so downstream skills can attach capture and delivery assets without falling back to Google Docs.

Recommended fields include:
- `Script URL`
- `Hand Raiser URL`
- `Hand Raiser Page`
- `HighLevel Funnel URL`
- `HighLevel Form URL`
- `HighLevel Calendar URL`
- `HighLevel Resource URL`
- `Google Doc URL (Legacy)` only if the user still wants it
