## HighLevel Instagram comment automation default

For comment-keyword CTAs, HighLevel is the default execution layer.

### Execution model
- use Instagram Comment(s) On A Post as the workflow trigger
- optionally reply in comments to confirm the request
- send the promised asset through Instagram Interactive Messenger or Instagram DM
- tag the contact and store the keyword, source post, and offer in HighLevel
- route the hand-raiser, funnel, booking link, or nurture sequence through HighLevel workflows

### CTA rule
Every final-slide comment keyword should map to a specific HighLevel workflow, not a generic external DM tool.
