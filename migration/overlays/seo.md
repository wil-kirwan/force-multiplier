## HighLevel-hosted site context

Treat HighLevel Sites, Funnels, website pages, landing pages, blogs, forms, surveys, calendars, and CRM conversion events as first-class SEO targets.

### When auditing HighLevel properties
- review SEO fields on site, funnel, and blog pages
- check forms, surveys, calendar bookings, thank-you pages, and conversion events
- inspect canonical, robots, Open Graph, schema, and custom code placement limitations
- use the shared `highlevel-api` skill to inspect related blogs, calendars, contacts, opportunities, and other assets when access exists
