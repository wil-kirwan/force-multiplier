## HighLevel-first proposal delivery

HighLevel Sites or Funnels are the default deployment path for this skill. HTML export remains optional.

### Default implementation
- publish the proposal as a HighLevel site page or funnel step
- attach a HighLevel form or calendar when the client needs to book, apply, or request next steps
- create or update the opportunity in the correct pipeline on submission or booking
- trigger HighLevel workflows for email, SMS, reminders, and follow-up

### Output rule
When the user asks for a proposal build, return the HighLevel deployment plan first. Treat Vercel or other hosting as optional export paths only.
