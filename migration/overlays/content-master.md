## Notion + HighLevel production pipeline

Keep Notion as the content database. Use HighLevel as the default capture, delivery, nurture, and CRM execution layer.

### Pipeline default
- scripts and research live in Notion
- lead magnets, funnels, forms, surveys, calendars, workflows, and CRM activity live in HighLevel
- comment-keyword CTAs use HighLevel Instagram comment automation and DM workflows
- follow-up uses HighLevel email, SMS, tags, custom fields, and opportunity routing

### Shared skill usage
If HighLevel auth and context are available, use the shared `highlevel-api` skill to create or update contacts, opportunities, blog posts, calendars, forms, workflows, Social Planner assets, or delivery URLs.
