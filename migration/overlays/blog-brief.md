## HighLevel CTA planning mode

Keep the research, outline, statistics, and distribution workflow unchanged. Add a HighLevel-first CTA and follow-up plan to every brief when the user is on HighLevel.

### CTA defaults
Prefer these destination types in this order:
1. HighLevel form or survey
2. HighLevel calendar booking
3. HighLevel funnel or website page
4. HighLevel email or SMS nurture entry point
5. HighLevel pipeline entry with workflow follow-up

### Add this field map to every HighLevel brief
- CTA destination
- tags to apply
- custom fields to capture
- pipeline and starting stage
- workflow that should fire
- follow-up channel, such as email, SMS, or both

If the user wants HighLevel Blogs output, also include slug, author, category, canonical, SEO title, post description, and cover-image requirements.
